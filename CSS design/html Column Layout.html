nsert Disc 1 into your CD/DVD drive. If you have Autorun enabled the game should start installing immediately. If Autorun is disabled, browse to the CD and double-click on 'setup.exe'. You will be prompted for your choice of language for the installer program - select it from the drop-down listYou will then be asked to enter the directory you wish to install the game to.After a few minutes you will then be asked to insert Disc 2 to continue the install. Swap Disc 1 for Disc 2 and select 'OK'.
